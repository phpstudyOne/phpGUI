GdkGravity

Defines the reference point of a window and the meaning of coordinates passed to move() . See move() and the "implementation notes" section of the extended window manager hints specification for more details.

Value

Symbolic name

Description

  1Gdk::GRAVITY_NORTH_WEST The reference point is at the top left corner.
  2Gdk::GRAVITY_NORTH The reference point is in the middle of the top edge.
  3Gdk::GRAVITY_NORTH_EAST The reference point is at the top right corner.
  4Gdk::GRAVITY_WEST The reference point is at the middle of the left edge.
  5Gdk::GRAVITY_CENTER The reference point is at the center of the window.
  6Gdk::GRAVITY_EAST the reference point is at the middle of the right edge.
  7Gdk::GRAVITY_SOUTH_WEST The reference point is at the lower left corner.
  8Gdk::GRAVITY_SOUTH The reference point is at the middle of the lower edge.
  9Gdk::GRAVITY_SOUTH_EAST The reference point is at the lower right corner.
  10Gdk::GRAVITY_STATIC The reference point is at the top left corner of the window itself, ignoring window manager decorations.
