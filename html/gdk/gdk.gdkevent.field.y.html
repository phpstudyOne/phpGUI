GdkEvent::y

Access: Read Write
Type: int

The y coordinate of the pointer relative to the window.

This field is available for the following events:

  • Gdk::MOTION_NOTIFY
  • Gdk::BUTTON_PRESS
  • Gdk::ENTER_NOTIFY
  • Gdk::CONFIGURE
  • Gdk::SCROLL
