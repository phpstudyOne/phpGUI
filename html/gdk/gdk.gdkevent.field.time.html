GdkEvent::time

Access: Read Write
Type: int

The time of the event in milliseconds.

This field is available for the following events:

  • Gdk::MOTION_NOTIFY
  • Gdk::BUTTON_PRESS
  • Gdk::KEY_PRESS
  • Gdk::ENTER_NOTIFY
  • Gdk::PROPERTY_NOTIFY
  • Gdk::SELECTION_CLEAR
  • Gdk::PROXIMITY_IN
  • Gdk::DRAG_ENTER
  • Gdk::SCROLL
  • Gdk::OWNER_CHANGE
