GdkEvent::device

Access: Read Only
Type: GdkDevice

The device where the event originated.

This field is available for the following events:

  • Gdk::MOTION_NOTIFY
  • Gdk::BUTTON_PRESS
  • Gdk::PROXIMITY_IN
  • Gdk::SCROLL
