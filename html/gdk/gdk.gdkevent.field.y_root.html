GdkEvent::y_root

Access: Read Write
Type: int

The y coordinate of the pointer relative to the root of the screen.

This field is available for the following events:

  • Gdk::MOTION_NOTIFY
  • Gdk::BUTTON_PRESS
  • Gdk::ENTER_NOTIFY
  • Gdk::DRAG_ENTER
  • Gdk::SCROLL
