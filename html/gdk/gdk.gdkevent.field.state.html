GdkEvent::state

Access: Read Only
Type: int

A bit-mask representing the state of the modifier keys (e.g. Control, Shift and Alt) and the pointer buttons.

This field is available for the following events:

  • Gdk::MOTION_NOTIFY
  • Gdk::BUTTON_PRESS
  • Gdk::KEY_PRESS
  • Gdk::ENTER_NOTIFY
  • Gdk::PROPERTY_NOTIFY
  • Gdk::VISIBILITY_NOTIFY
See also: GdkModifierType
