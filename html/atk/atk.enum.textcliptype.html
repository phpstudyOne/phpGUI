AtkTextClipType

Text boundary types used for specifying boundaries for regions of text

Value

Symbolic name

Description

  0Atk::TEXT_CLIP_NONE No clipping to be done
  1Atk::TEXT_CLIP_MIN Text clipped by min coordinate is omitted
  2Atk::TEXT_CLIP_MAX Text clipped by max coordinate is omitted
  3Atk::TEXT_CLIP_BOTH Only text fully within mix/max bound is retained
