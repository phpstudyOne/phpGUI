GtkWidget

Base class for all widgets.

Object Hierarchy

GObject
`-- GtkObject
       `-- GtkWidget

Direct Subclasses

GtkCalendar, GtkCellView, GtkContainer, GtkDrawingArea, GtkEntry, GtkInvisible, GtkMisc, GtkOldEditable, GtkPreview, GtkProgress, GtkRange, GtkRuler, GtkSeparator, GtkPlot

Description

A widget is any class which inherits from GtkWidget. Widgets are classes which can listen for and react to user events. Widgets are usually associated with display and/or layout.
