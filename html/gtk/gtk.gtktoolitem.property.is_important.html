GtkToolItem::is-important

Access: Read Write
Type: bool

Whether the toolbar item is considered important. When true, toolbar buttons show text in Gtk::TOOLBAR_BOTH_HORIZ mode.

Default is false.

See also: get_is_important, set_is_important
