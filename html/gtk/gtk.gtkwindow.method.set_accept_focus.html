GtkWindow::set_accept_focus

void set_accept_focus(bool accept);

Sets whether or not the window will accept input focus. If accept is true, the window will accept focus. If it is false, it will not.

This method only sets a hint that the window manager can use to decide if the window should be given focus. You should not rely 100% on this method preventing the window from receiving focus.

See also: get_accept_focus()
