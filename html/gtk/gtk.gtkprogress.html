GtkProgress

A base class for GtkProgressBar.

Object Hierarchy

GObject
`-- GtkObject
       `-- GtkWidget
              `-- GtkProgress

Direct Subclasses

GtkProgressBar

Description

A GtkProgress is the abstract base class used to derive a GtkProgressBar which provides a visual representation of the progress of a long running operation.

Properties
