GtkCellRendererText Constructor

GtkCellRendererText ();

Creates a new GtkCellRendererText. The way that text is drawn is changed using object properties. The object properties can be set globally (with set_property() ). Also, with GtlTreeViewColumn, you can bind a property to a value in a GtkTreeModel. For example, you can bind the "text" property on the cell renderer to a string value in the model, thus rendering a different string in each row of the GtkTreeView.
