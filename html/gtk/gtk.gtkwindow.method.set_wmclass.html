GtkWindow::set_wmclass

void set_wmclass(string wmclass_name, string wmclass_class);

Sets the window manager name and class hints for the window to wmclass_name and wmclass_class.

This method should not be called in user code. Gtk+ takes care of setting the name and class automatically.
