GtkWindow::type_hint

Access: Read Only
Type: GdkWindowTypeHint

A hint to help the window manager determine the window type so that it can treat the window properly.

Defaults to Gdk::WINDOW_TYPE_HINT_NORMAL.

See also: get_type_hint() , set_type_hint()
