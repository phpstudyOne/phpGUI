GtkExpanderStyle

Used to specify the style of the expanders drawn by a GtkTreeView.

Value

Symbolic name

Description

  0Gtk::EXPANDER_COLLAPSED The style used for a collapsed subtree.
  1Gtk::EXPANDER_SEMI_COLLAPSED Intermediate style used during animation.
  2Gtk::EXPANDER_SEMI_EXPANDED Intermediate style used during animation.
  3Gtk::EXPANDER_EXPANDED The style used for an expanded subtree.
