move-handle

The "move-handle" signal is emitted when paned has the focus and one of the following key combinations is pressed to move the separator:

  • Left Arrow
  • Right Arrow
  • Up Arrow
  • Down Arrow
  • Page Up
  • Page Down
  • Home
  • End

Callback function

bool callback(GtkPaned paned, GtkScrollType scrolltype);
