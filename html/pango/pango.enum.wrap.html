PangoWrap

A PangoWrapMode describes how to wrap the lines of a PangoLayout to the desired width.

Value

Symbolic name

Description

  0Pango::WRAP_WORD Wrap lines at word boundaries.
  1Pango::WRAP_CHAR Wrap lines at character boundaries.
  2Pango::WRAP_WORD_CHAR Wrap lines at word boundaries, but fall back to character boundaries if there is not enough space for a full word.
