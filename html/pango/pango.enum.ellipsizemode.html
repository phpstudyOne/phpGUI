PangoEllipsizeMode

The PangoEllipsizeMode type describes what sort of (if any) ellipsization should be applied to a line of text. In the ellipsization process characters are removed from the text in order to make it fit to a given width and replaced with an ellipsis.

Value

Symbolic name

Description

  0Pango::ELLIPSIZE_NONE No ellipsization.
  1Pango::ELLIPSIZE_START Omit characters at the start of the text.
  2Pango::ELLIPSIZE_MIDDLE Omit characters in the middle of the text.
  3Pango::ELLIPSIZE_END Omit characters at the end of the text.
